# Ink & Iron Tattoo Studio

Modern, dark-themed studio website with:
- Hero-focused homepage and artistic visual styling
- Filterable tattoo/piercing gallery
- Services and pricing section
- Artist profile section
- Booking request form with client-side validation
- About/values section focused on hygiene and safety
- Contact section with map and social links

## Run locally

```bash
python app.py
```

Then open `http://localhost:8000`.
